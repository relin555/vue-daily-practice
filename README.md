# app-practice

## Project setup
```
ежедневная практика на vue

для запкска проекта скачайте загрузите его к себе на диск:

$ git fetch https://github.com/relin555/vue-daily-practice.git

установите пакеты:

npm install

зайдите в папку проекта и запустите проект 

npm run serve

```
